**Таблица на типовите елементи в Scala**



|           Име           | Синтаксис |                            Пример                            |                           Бележки                            |
| :---------------------: | :-------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|      ***Типове***       |     -     |                              -                               |                              -                               |
|      Номинален тип      | `String`  |                     `"Viktor"`, `"hi!"`                      |                                                              |
